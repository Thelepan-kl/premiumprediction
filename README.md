
# Insurance Premium Prediction

A machine learning web application that predicts insurance premium charges based on personal health and demographic details like age, BMI, smoking status, and region.

## Tech Stack
- Python, Scikit-learn, Flask
- Ridge Regression with Standard Scaler
- HTML, CSS
- GitHub Actions (CI/CD)

## How to Run

```bash
git clone https://github.com/Thelepan-kl/premiumprediction.git
cd premiumprediction
pip install -r requirements.txt
python app.py
```

Open `http://localhost:5000` in your browser.

## Project Structure
- app.py                    -- Flask application
- premium_prediction.ipynb  -- Model training notebook
- insurance.csv             -- Dataset
- regmodel.pkl              -- Trained model
- scaling.pkl               -- Standard Scaler
- requirements.txt
